# Words and phrases to avoid

Words, expressions, and constructions overused by AI in English. Each entry includes natural alternatives.

These aren't absolute bans. If a listed word is genuinely the most precise choice in context, use it. What we're avoiding is the automatic, recurring, disproportionate use that signals machine-generated text.

## Hard-banned words

These words are so strongly associated with AI output that they should be avoided entirely. There's always a better alternative.

| Banned | Alternatives |
|--------|-------------|
| delve | explore, look at, examine, dig into |
| moreover | (cut it — just continue the thought) |
| furthermore | also, and, plus |
| albeit | though, even though, although |
| indeed | (cut it — strengthen the sentence itself) |
| certainly | (cut it or use "yes") |
| utilize | use |
| leverage (verb) | use, take advantage of, build on |
| facilitate | help, enable, make easier |
| tapestry | (cut it — describe specifically) |
| landscape (metaphorical) | field, area, world, scene |
| foster | encourage, support, grow |
| underscore | show, highlight, prove |
| testament | proof, sign, evidence |
| nuanced | (be specific about what the nuance is) |
| multifaceted | complex, varied (or describe the facets) |
| myriad | many, lots of, countless |
| plethora | many, lots of, plenty |
| pivotal | important, key, turning point |
| paramount | most important, top priority |
| encompass | include, cover |
| commenced | started, began |
| endeavor | effort, try, attempt |
| aforementioned | (name the thing again, or use "this") |
| henceforth | from now on |

## Throat-clearing openers

Predictable opening phrases. Cut them and start with the actual content.

| Avoid | What to do |
|-------|-----------|
| In today's fast-paced world... | Start with the specific claim |
| In the ever-evolving landscape of... | Name the field and say what changed |
| It's worth noting that... | Just note it |
| It's important to understand that... | Say the thing |
| Let's dive in / Let's dive into... | Start directly |
| Let's break it down | Start breaking it down |
| Let's unpack this | Start unpacking |
| Let me be clear | Be clear without announcing it |
| Here's the thing | State the thing |
| The reality is... | Say the reality |
| The truth is... | Say the truth |
| When it comes to... | Name the subject directly |
| As we all know... | (cut it — condescending) |
| It goes without saying... | (if it goes without saying, don't say it) |
| In order to... | To... |
| At its core... | (cut it) |
| At the end of the day... | (cut it) |
| Great question! | (cut it — just answer) |
| That's a great point | (cut it — respond to the point) |

## Emphasis crutches

Phrases AI adds for weight that carry no information.

| Avoid | What to do |
|-------|-----------|
| Let that sink in. | (cut it — trust the reader) |
| Full stop. | (cut it — the period already did that) |
| Period. | (same as above) |
| This matters because... | (show why it matters through the content) |
| And that's okay. | (cut it unless genuinely comforting) |
| I'm going to be honest... | Be honest without announcing it |
| The uncomfortable truth is... | State the truth |
| Here's what nobody talks about... | Talk about it |
| This is huge. | (cut it or be specific about the size) |
| Think about it. | (cut it — they're already reading) |
| Read that again. | (cut it) |
| This cannot be overstated. | (cut it — the content should speak) |

## Wrap-up clichés

Phrases AI puts at the end of sections or articles as unnecessary summaries.

| Avoid | What to do |
|-------|-----------|
| In conclusion... | End with the final idea, don't announce it |
| To sum up... | Only in very long texts where it genuinely helps |
| In summary... | Same as above |
| Ultimately... | (cut it or rephrase) |
| The bottom line is... | Just state it |
| Moving forward... | (cut it) |
| The future looks bright. | (cut it — empty optimism) |
| Only time will tell. | (cut it — says nothing) |
| The possibilities are endless. | (cut it — they aren't) |
| As we've seen... | (cut it — the reader was there) |
| All in all... | (cut it) |

## Formal connectors overused by AI

Natural English uses connectors, but AI overuses the most formal, academic ones.

| Overused | Natural alternatives |
|----------|---------------------|
| Moreover | also, and, plus (or just start the next sentence) |
| Furthermore | also, and, on top of that |
| Nevertheless | but, still, even so |
| Consequently | so, as a result, that's why |
| Subsequently | then, after that, next |
| In this regard | (cut it — usually unnecessary) |
| Conversely | on the other hand, but |
| Notwithstanding | despite, even though |
| Thus | so |
| Hence | so, that's why |
| Thereby | (restructure the sentence) |
| Wherein | where, in which |
| Additionally | also, and, plus |

## Significance inflation

Adjectives and expressions AI uses to pump up the importance of any topic.

| Inflated | Alternatives |
|----------|-------------|
| revolutionary | new, different, major |
| unprecedented | new, first-time, never-before-seen (only if literally true) |
| game-changing | important, significant |
| transformative | important, impactful (describe the impact) |
| extraordinary | very good, notable, impressive |
| remarkable | good, notable, worth mentioning |
| exceptional | very good, outstanding |
| invaluable | very useful, essential |
| indispensable | essential, necessary |
| unparalleled | best, top, leading |

## Overused adverbs

AI leans on adverbs that add little meaning.

| Avoid | What to do |
|-------|-----------|
| significantly | a lot, much, considerably (or use a number) |
| fundamentally | at its core, basically (or cut it) |
| essentially | basically (or cut it) |
| incredibly | very, extremely (or cut it and strengthen the noun) |
| particularly | especially, mainly |
| undeniably | (cut it — deny it if you can, otherwise just state it) |
| inherently | by nature, built-in |
| genuinely | really, truly (or cut it) |
| arguably | (cut it — make the argument or don't) |
| literally | (cut it unless something is literally literal) |
| simply | just (or cut it) |
| merely | just, only |

## Hedging phrases

AI hedges excessively to avoid being wrong. Cut these and commit to a statement.

| Avoid | What to do |
|-------|-----------|
| It could potentially be argued that... | (make the argument or don't) |
| It's perhaps worth considering... | Consider this: |
| One might suggest that... | (say it directly) |
| It's not unreasonable to think... | (say what you think) |
| While specific details are limited... | (be specific about what you know) |
| Based on available information... | (just state the information) |

| drive (results) | produce, create, lead to, cause |
| ensure | make sure, guarantee, confirm |
| enhance | improve, boost, strengthen |
| bolster | support, strengthen, back up |
| foster | grow, encourage, build |
| mitigate | reduce, lessen, limit |
